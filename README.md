# README #

Shopping Cart with React Redux

### FEATURES ###

* Product List
* Add to cart functionality
* Add/Remove/Increase/Decrease Quantity

### How do I get set up? ###

* Clone the repo
* npm i
* npm run build [for prod]
* npm run build:dev [for dev]
* npm start
* npm test [for tests]
